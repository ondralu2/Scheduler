# Scheduler
Application for schedule meetings with friends or other people.

Úkolem projektu je vytvořit softwarovou aplikaci pro zjednodušení plánování skupinových událostí. 
Umožňuje vytvoření události s různými časovými termíny a/nebo místy, poté si pozvaní uživatelé na událost 
vyberou preferované termíny a/nebo místa pomocí formuláře, svoji volbu uloží a autor události 
zvolí ideální termín a/nebo místo na základě volby všech pozvaných uživatelů. 
Je možné vybrat pouze termíny, pouze místa nebo obojí. Hlavním prvkem aplikace jsou tedy události, 
které obsahují textový popis, libovolný počet termínů a míst (podle preferencí autora) a seznam přihlášených uživatelů na konkrétní termíny a místa. 
Počet uživatelem preferovaných termínů a míst není omezen. Upravovat událost (včetně termínů a míst) může pouze její autor. 
Uživatel zadává do aplikace základní informace o sobě – jméno, příjmení, telefon a email. 
Termíny jsou uvedeny jako datum a čas, u míst je uvedena adresa, název podniku nebo místa a zeměpisné souřadnice.
